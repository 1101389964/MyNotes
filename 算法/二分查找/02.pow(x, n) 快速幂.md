## [Pow(x, n)](https://leetcode.cn/problems/powx-n/)

* **题目：**

  >实现 [pow(*x*, *n*)](https://www.cplusplus.com/reference/valarray/pow/) ，即计算 `x` 的整数 `n` 次幂函数（即，`xn` ）。

* **示例：**

  * **示例1：**

    ```
    输入：x = 2.00000, n = 10
    输出：1024.00000
    ```

  * **示例2：**

    ```
    输入：x = 2.10000, n = 3
    输出：9.26100
    ```

  * **示例3：**

    ```
    输入：x = 2.00000, n = -2
    输出：0.25000
    解释：2-2 = 1/22 = 1/4 = 0.25
    ```

  * **提示：**

  * 

    * `-100.0 < x < 100.0`
    * `-231 <= n <= 231-1`
    * `-10^4 <= x^n <= 10^4`

* **解析：**

  >举个栗子：
  >
  >对于偶数次方：
  >
  >* x^8 = x^4 * x^4
  >* x^4 = x^2 * x^2
  >* x^2 = x * x
  >
  >对于奇数次方：
  >
  >* x^7 = x * x^3 * x^3
  >* x^3 = x * x^2
  >* x^2 = x * x

* **代码：**

  ```js
  // 分治
  var myPow = function(x, n) {
      // x == 1 或 n == 0 直接返回 1
      if (x === 1 || n === 0) {
          return 1;
      }else if(n > 0) {
          return foo(x, n);
      }else {
          // 当 n < 0 需要返回倒数
          return 1 / foo(x, -n);
      }
  };
  
  function foo(x, n) {
      if (n === 1) {
          return x;
      }else if (n % 2 === 0) {
          const res1 = foo(x, n / 2);
          return res1 * res1;
      }else {
          const res2 = foo(x, (n / 2) >> 0)
          return res2 * res2 * x;
      }
  }
  ```
