## [替换空格](https://leetcode.cn/problems/ti-huan-kong-ge-lcof/)

* **题目：**

  >请实现一个函数，把字符串 `s` 中的每个空格替换成"%20"。

* **示例：**

  * **示例1：**

    ```
    输入：s = "We are happy."
    输出："We%20are%20happy."
    ```

  * **提示：**

    * 0 <= s 的长度 <= 10000

* **解析：**

  >

* **代码：**

  ```js
  /**
   * @param {string} s
   * @return {string}
   */
  const replaceSpace = function(s){
      let result = '';
      for(let i = 0; i < s.length; i++){
          if(s[i] !== ' '){
              result += s[i];
          }else{
              result += '%20';
          }
      }
      return result;
  }
  ```
